Nanak Pet Store is a modern, responsive pet shop website built using HTML, CSS, and JavaScript.
It provides a smooth shopping experience for pet lovers with a clean UI, product browsing, and cart functionality.
This project demonstrates strong frontend development skills, including layout design, interactivity, and responsive web design.

Features
🏠 Website Features
Multi-section landing page (Home, About, Services, Shop, Contact)
Beautiful hero section with modern UI design
Responsive navigation menu (desktop + mobile)

🛒 E-commerce Features
Product listing system (pet products/services)
Add to cart functionality
Shopping cart sidebar with:
Add/remove items
Quantity control
Total calculation
Cart overlay system

🎨 UI/UX Features
Modern responsive design
Smooth animations and transitions
Toast notifications for actions
Modal popups (alerts/feedback)
Star rating system
Interactive buttons and hover effects

Mobile Friendly
Fully responsive design for all screen sizes
Mobile navigation menu (hamburger menu)
Optimized layout for phones and tablets

🛠️ Technologies Used
HTML5
CSS3 (Flexbox + Grid)
JavaScript (DOM manipulation)
Google Fonts
